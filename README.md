# Tutor-grasp (Under Construction)

A **Tuition Management System** to efficiently manage students, teachers, parents, attendance, fees, and homework.  
This project is **currently under development**, and features are being gradually added.

## 🌟 Features (Planned / Partial)

- **Dashboard** for teachers, and parents  
- **Student Management:** Add, edit, delete, view student details  
- **Parent Management:** Parent authentication and access to child’s progress  
- **Attendance Tracking:** Record and view student attendance  
- **Fees Management:** Track payments, due fees, and payment history  
- **Homework Management:** Assign and track homework submissions  
- **Authentication:** Separate login for **Teachers** and **Parents**  

> ⚠️ **Note:** The project is under construction. Some features may not be fully functional yet.

## 🛠️ Technologies Used

- **Frontend:** React.js, HTML, CSS, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB / MySQL (depending on implementation)  
- **Tools:** Postman for API testing, VS Code for development
